# vedic_panchanga_dart Example

This directory contains examples demonstrating how to use the vedic_panchanga_dart package.

## Running the Example

```bash
dart run example/example.dart
```

## What the Example Demonstrates

The example file shows:

1. **Complete Panchanga Calculation** - Get all panchanga elements for a day
2. **Individual Element Calculations** - Calculate specific items like sunrise, tithi, nakshatra
3. **Planet Positions** - Get positions of all 9 planets
4. **Different Ayanamsa Modes** - Compare results using different ayanamsa systems
5. **JSON Export** - Export panchanga data as JSON
6. **Multiple Days** - Calculate panchanga for a week

## Expected Output

The example will print:
- Complete panchanga details for Chennai
- Sunrise/sunset times for Mumbai
- Planetary positions for Delhi
- Comparison of moon positions using different ayanamsas
- Weekly tithi information

## Customization

You can modify the example to:
- Use your own location coordinates
- Change the date range
- Try different ayanamsa modes
- Export data in different formats
