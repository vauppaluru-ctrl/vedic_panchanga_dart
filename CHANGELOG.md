## 0.1.0

* Initial release
* Core panchanga calculations:
  * Sunrise, sunset, moonrise, moonset
  * Tithi (lunar day)
  * Nakshatra (lunar mansion) with pada
  * Yoga
  * Karana
  * Moon's Rasi (zodiac sign)
* Planetary positions (Sun to Ketu)
* Multiple ayanamsa modes (LAHIRI, KP, RAMAN, TRUE_CITRA, etc.)
* High-precision calculations using Swiss Ephemeris
* JSON export support
* Complete data models for all panchanga elements
